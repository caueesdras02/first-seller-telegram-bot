# 🤖 First Seller - Bot de Vendas no Telegram

Bot desenvolvido em Python utilizando a API do Telegram, com o objetivo de simular um atendimento automatizado para vendas.

## 🎯 Objetivo
Praticar automação, integração com APIs e desenvolvimento de bots.

## 🧩 Funcionalidades
- Respostas automáticas
- Simulação de atendimento
- Fluxo de vendas

## 🛠️ Tecnologias utilizadas
- Python
- API do Telegram
